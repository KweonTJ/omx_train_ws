## 동작 범위

```mermaid
flowchart LR
    A[물체 접근] --> B[그리퍼 Close]
    B --> C[물체를 들고 Stay 복귀]
    C -. 베이스 배송 이동 .-> D[배송 위치 전개]
    D --> E[그리퍼 Open / 물체 해제]
    E --> F[빈 팔 Stay 복귀]
```

- ArUco 기반 물체 좌표를 로봇 `base_link` 기준 목표로 사용
- 파지 전까지 목표 좌표를 갱신해 팔 경로 보정
- 파지·배송 episode를 분리해 베이스 주행은 학습에서 제외
- 위치, yaw, 타워 높이, 물체 크기, 동역학, 마찰, Vision 오차를 episode마다 랜덤화

## 해석 범위

- 현재 물체 형상은 직육면체 계열이며 크기 변동만 학습함
- 원통, 봉투, 연성 물체, 비대칭 물체의 파지는 검증하지 않음
- 시뮬레이션 성공률은 실기기 성공률을 직접 의미하지 않음
- 실제 ArUco 오차와 Dynamixel 지연 로그를 반영한 후 최종 배포 모델을 다시 선정해야 함
